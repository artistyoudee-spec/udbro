📌 Google Drive Link Setup (Important!)

  1. Upload your file to Google Drive
  2. Right-click the file → Share → Change to "Anyone with the link" → Set role to "Viewer"
  3. Click "Copy link"
  4. Paste the link in the Media URL field below

Supported link formats:

  • https://drive.google.com/file/d/FILE_ID/view?usp=sharing
  • https://drive.google.com/open?id=FILE_ID
  • Any Google Drive sharing URL

⚠️ The file MUST be shared as "Anyone with the link" — otherwise it won't load on the website!
